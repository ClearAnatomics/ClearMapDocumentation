Stitching¶

ClearMap.Alignment.Stitching.StitchingRigid

Stitching module for aligning and stitching data sets ridgidly.

ClearMap.Alignment.Stitching.StitchingWobbly

Wobbly stitching module handles the alginment of large volumetric data sets.

ClearMap.Alignment.Stitching.Tracking

Simple particle tracker based on liner programming.

Stitching module for aligning and stitching data sets with ClearMap.

Stitching can be done in two ways:

  • rigidly via the StitchingRigid or

  • wobbly via StitchingWobbly.
